# Federated Learning for Medical Imaging

## Overview
This project explores privacy-preserving deep learning for medical imaging using Federated Learning (FL). The system simulates multiple healthcare clients training a shared model without exchanging raw patient data.

## Features
- Federated client-server architecture
- Privacy-preserving AI training
- Medical imaging classification
- PyTorch + Flower framework
- Simulated decentralized hospitals

## Research Motivation
Traditional AI systems require centralized datasets, which can compromise patient privacy. Federated Learning enables collaborative training while keeping sensitive medical data local.

## Tech Stack
- Python
- PyTorch
- Flower (FL)
- NumPy
